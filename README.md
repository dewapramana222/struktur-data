Basis kode ini digunakan untuk mendefinisikan sebuah array (list) kosong yang akan digunakan untuk menyimpan data nila mahasiswa, disini saya megunakan kalimat nilai_mahasiswa swbagai list. 
Tanda # hanya untuk menujukan komentar penjelas.
nilai_mahasiswa = nama variabel untuk menyimpan data nilai mahasiswa dan memudahkan pengolahan data.
= → operator penugasan untuk memberikan nilai ke variabel.
